COVID-CT-DS is a comprehensive dataset designed to advance AI research in medical image analysis for COVID-19. It comprises 278 patient-specific CT cases with varying disease severity (normal: 10, mild:88, moderate:112, severe:48, fibrosis:16). Each patient-specific HRCT case comprises of an average of 550 axial CT images. For each case, a diagnostic report is also provided having detailed information. By providing a large-scale, well-balanced resource, this dataset aims to facilitate the development of AI models capable of accurately assessing disease severity and aiding clinical decision-making.
